Para atualizar o dotnet-ef, o caminho mais comum é:

Ferramenta global:
dotnet tool update --global dotnet-ef
Se ainda não estiver instalado:
dotnet tool install --global dotnet-ef
Conferir versão:
dotnet ef --version
